About

Welcome to PSE Basic Financials, a website built using Python Django web framework.

As a self-taught developer, this project was an opportunity for me to put my skills to the test and create a valuable resource for others. I wanted to create a platform that makes it easy for people to access and understand financial data of listed companies in the Philippines.

I believe that having access to comprehensive financial data is important and that's why I created this website. My goal is to provide a user-friendly platform that makes it easy to browse and search financial data and help investors make informed decisions.

I am constantly learning and looking for ways to improve the site, so feel free to reach out to me with any feedback or suggestions.

I am also proud to say that PSE Basic Financials is an open source project. You can find the codebase on the icon below and contribute to the development of the website if you are interested in doing so.
